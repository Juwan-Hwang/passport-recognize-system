# 出入境证件鉴别与研判系统 [BCIS-V15]

## 1. 项目简介
**BCIS (Border Control Intelligence System)** 是一套专为海关、边检及执法部门设计的**离线应急查验工具**。该系统完全在浏览器端运行（零网络请求），用于在断网、设备故障或应急环境下，快速解析、验证和研判国际旅行证件的机读区（MRZ）数据。

系统不仅支持标准的 ICAO 9303 格式，还针对**中国护照**、**港澳台通行证**以及全球 20+ 个主要国家的身份证件进行了深度优化，能够挖掘隐藏在“可选数据域”中的行政编号和个人敏感信息，并提供自动化的风险评估。

---

## 2. 核心功能

### 🛡️ 100% 离线安全架构
*   **纯前端运行**：基于 React + TypeScript 构建，无后端服务，无数据上传。
*   **数据主权**：所有敏感数据仅保留在浏览器内存中，刷新即焚，完全符合保密环境（Air-gapped）部署要求。

### 🌏 全球证件标准支持
*   **TD3 (护照)**: 标准 2行 x 44字符 (如: 护照, 签证 MRV-A)。
*   **TD2 (旧版身份证/签证)**: 标准 2行 x 36字符 (如: 德国旧版身份证, 签证 MRV-B)。
*   **TD1 (卡式证件)**: 标准 3行 x 30字符 (如: 欧盟/美国卡式证件)。
*   **CN_CARD (中国特色证件)**: 专有单行 30字符格式 (如: 往来港澳通行证, 往来台湾通行证)。

### 🔍 深度数据挖掘与情报研判
系统内置了针对特定国家的“反向工程”逻辑，可从 MRZ 的可选域中提取隐藏信息：
*   **🇨🇳 中国 (CHN)**:
    *   **GBK 隐形姓名解析**: 从电子护照的加密域中还原中文姓名。
    *   **拼音交叉验证**: 自动对比 MRZ 拼音与中文姓名，检测伪造的姓名截断标记（防篡改逻辑）。
*   **🇩🇪 德国 (DEU)**: 解析身份证的各行政区代码及序列号逻辑。
*   **🇪🇸 西班牙 (ESP)**: 提取 DNI 身份证号码。
*   **🇵🇱 波兰 (POL)**: 提取 PESEL 公民身份号。
*   **🇳🇱 荷兰 (NLD)**: 提取 BSN 公民服务号。
*   **以及**: 法国、比利时、瑞士、罗马尼亚、保加利亚、乌克兰、俄罗斯、北欧四国、以色列等 20+ 地区的特有逻辑。

### ⚠️ 自动化风险评估
系统会自动计算并标注以下风险：
*   **数据完整性**: 全字段 Modulus-10 校验位验证。
*   **证件状态**: 过期提醒、临期预警。
*   **逻辑漏洞**: 检测伪造证件常见的逻辑错误（如截断标记与剩余空间不符、元音数量不足以构成声称的汉字数量等）。

### 💻 执法级可视化界面
*   **终端风格视图**: 对 MRZ 原始数据进行高亮分色显示（签发国、姓名、证件号、校验位）。
*   **中英双语**: 支持一键切换界面语言，适应跨国联合执法场景。

*   **核心框架**: React 19

*   **辅助库**: `pinyin-pro` (用于中文姓名拼音的高精度比对)

---

## 4. 部署与使用

### 环境要求
*   现代浏览器（Chrome, Edge, Firefox）。
*   无需互联网连接。

### 操作指南
1.  **输入数据**: 在主界面的输入框中，通过扫码枪或手工输入证件的 MRZ 码（支持自动去除空格和换行）。
2.  **执行查验**: 点击“执行查验”按钮或按回车。
3.  **查看报告**:
    *   **状态栏**: 显示校验结果、有效期状态及风险等级。
    *   **身份信息**: 显示解析出的姓名、性别、生日等。如果是中国护照，尝试显示 GBK 解码的中文名。
    *   **技术参数**: 显示证件细分类型（精确到国别和版式）。
    *   **数据透视**: 查看底层的校验位计算逻辑。

## 5. 证件细分类型支持列表（部分）
系统能精确识别并分类以下证件：
*   **大中华区**: 中华人民共和国普通/电子护照、香港特区护照、澳门特区护照、往来港澳/台湾通行证。
*   **美洲**: 美国护照/护照卡、加拿大护照、巴西护照。
*   **欧洲**: 德国、法国、英国、意大利、西班牙、荷兰、波兰、瑞士、俄罗斯、乌克兰等国护照及身份证。
*   **亚太**: 日本、韩国、澳大利亚、新西兰、新加坡、马来西亚、泰国、印度等。
