## 1. 프로젝트 개요
**BCIS (Border Control Intelligence System)** 는 **오프라인 검증 도구**입니다. 본 시스템은 완전히 브라우저에서 실행되며(네트워크 요청 제로), 국제 여행 증명서의 기계 판독 영역(MRZ) 데이터를 신속하게 파싱, 검증 및 평가합니다.

표준 ICAO 9303 형식을 지원할 뿐만 아니라, **중국 여권**, **홍콩·마카오·대만 통행증** 및 전 세계 20개 이상 주요 국가의 신분증에 대해 심층 최적화를 진행했습니다. "선택적 데이터 필드"에 숨겨진 행정 번호와 개인 민감 정보를 추출하고 자동화된 위험 평가를 제공합니다.

---

## 2. 핵심 기능

### 🛡️ 100% 오프라인 보안 아키텍처
*   **순수 프론트엔드 실행**: React + TypeScript로 구축, 백엔드 서비스 없음, 데이터 업로드 없음.
*   **데이터 주권**: 모든 민감 데이터는 브라우저 메모리에만 보관되며, 새로고침 시 소멸. 에어갭(Air-gapped) 환경 배포 요구사항에 완전 준수.

### 🌏 글로벌 증명서 표준 지원
*   **TD3 (여권)**: 표준 2행 × 44자 (예: 여권, MRV-A 비자).
*   **TD2 (구형 신분증/비자)**: 표준 2행 × 36자 (예: 독일 구형 신분증, MRV-B 비자).
*   **TD1 (카드형 증명서)**: 표준 3행 × 30자 (예: EU/미국 카드형 증명서).
*   **CN_CARD (중국 특수 증명서)**: 전용 단일행 30자 형식 (예: 홍콩·마카오·대만 통행증).

### 🔍 심층 데이터 마이닝 및 정보 분석
국가별 특화된 "역공학" 로직을 내장하여 MRZ 선택 영역에서 숨겨진 정보 추출:
*   **🇨🇳 중국 (CHN)**:
    *   **GBK 숨김 이름 파싱**: 전자여권의 암호화 영역에서 중국어 이름 복원.
    *   **병음 교차 검증**: MRZ 병음과 중국어 이름을 자동 비교하여 위조 절단 마커 탐지 (변조 방지 로직).
*   **🇩🇪 독일 (DEU)**: 신분증의 각 행정구역 코드 및 일련번호 로직 파싱.
*   **🇪🇸 스페인 (ESP)**: DNI 신분증 번호 추출.
*   **🇵🇱 폴란드 (POL)**: PESEL 시민 신분 번호 추출.
*   **🇳🇱 네덜란드 (NLD)**: BSN 시민 서비스 번호 추출.
*   **기타**: 프랑스, 벨기에, 스위스, 루마니아, 불가리아, 우크라이나, 러시아, 북유럽 4개국, 이스라엘 등 20개 이상 지역의 고유 로직.

### ⚠️ 자동화 위험 평가
시스템은 다음 위험을 자동으로 계산 및 표시:
*   **데이터 무결성**: 전체 필드 Modulus-10 체크 디지트 검증.
*   **증명서 상태**: 만료 알림, 만료 임박 경고.
*   **논리적 취약점**: 위조 증명서에 흔한 논리 오류 탐지 (예: 절단 마커와 남은 공간 불일치, 주장하는 한자 수를 구성하기에 불충분한 모음 수 등).

### 💻 시각화 인터페이스
*   **터미널 스타일 뷰**: MRZ 원시 데이터를 하이라이트 색상 표시 (발행국, 이름, 증명서 번호, 체크 디지트).
*   **중영 이중언어**: 원클릭 언어 전환으로 국제 공동 법 집행 시나리오 대응.

## 3. 기술 스택
*   **핵심 프레임워크**: React 19
*   **언어**: TypeScript (엄격한 타입 안전성)
*   **스타일링**: Tailwind CSS (반응형 산업 디자인)
*   **보조 라이브러리**: `pinyin-pro` (중국어 이름 병음 고정밀 비교용)

## 4. 배포 및 사용

### 시스템 요구사항
*   모던 브라우저 (Chrome, Edge, Firefox).
*   인터넷 연결 불필요.

### 작동 가이드
1.  **데이터 입력**: 메인 인터페이스 입력란에 스캐너 또는 수동으로 증명서의 MRZ 코드 입력 (공백 및 줄바꿈 자동 제거).
2.  **검증 실행**: "검증 실행" 버튼 클릭 또는 Enter 키 입력.
3.  **보고서 확인**:
    *   **상태 표시줄**: 검증 결과, 유효기간 상태, 위험 수준 표시.
    *   **신원 정보**: 파싱된 이름, 성별, 생년월일 등 표시. 중국 여권의 경우 GBK 디코딩된 중국어 이름 표시 시도.
    *   **기술 파라미터**: 증명서 세부 유형 표시 (국가 및 형식까지 정밀).
    *   **데이터 인사이트**: 기본 체크 디지트 계산 로직 표시.

## 5. 지원 증명서 유형 목록 (일부)
시스템은 다음 증명서를 정확하게 식별 및 분류 가능:
*   **대중화권**: 중화인민공화국 일반/전자여권, 홍콩 특별행정구 여권, 마카오 특별행정구 여권, 홍콩·마카오·대만 통행증.
*   **아메리카**: 미국 여권/여권카드, 캐나다 여권, 브라질 여권.
*   **유럽**: 독일, 프랑스, 영국, 이탈리아, 스페인, 네덜란드, 폴란드, 스위스, 러시아, 우크라이나 등의 여권 및 신분증.
*   **아시아태평양**: 일본, 한국, 호주, 뉴질랜드, 싱가포르, 말레이시아, 태국, 인도 등.
