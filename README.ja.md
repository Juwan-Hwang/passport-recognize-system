## 1. プロジェクト概要
**BCIS (Border Control Intelligence System)** は、**オフライン検証ツール**です。本システムは完全にブラウザ上で動作し（ネットワークリクエストゼロ）、国際旅行証明書の機械読取領域（MRZ）データの高速解析、検証、評価を行います。

標準的な ICAO 9303 形式に対応するだけでなく、**中国パスポート**、**香港・マカオ・台湾渡航証**、および世界20ヶ国以上の主要国の身分証明書に対して深度最適化を実施。「オプションデータフィールド」に隠された行政番号や個人情報を抽出し、自動リスク評価を提供します。

---

## 2. コア機能

### 🛡️ 100% オフライン セキュリティアーキテクチャ
*   **純粋フロントエンド実行**: React + TypeScript で構築、バックエンドサービス不要、データアップロードなし。
*   **データ主権**: すべての機密データはブラウザのメモリ内にのみ保持され、リフレッシュで消去。エアギャップ環境の展開要件に完全準拠。

### 🌏 グローバル証明書標準サポート
*   **TD3（パスポート）**: 標準 2行 × 44文字（例：パスポート、MRV-A ビザ）。
*   **TD2（旧型身分証/ビザ）**: 標準 2行 × 36文字（例：ドイツ旧型身分証、MRV-B ビザ）。
*   **TD1（カード型証明書）**: 標準 3行 × 30文字（例：EU/米国カード型証明書）。
*   **CN_CARD（中国独自証明書）**: 専用単一行 30文字形式（例：香港・マカオ・台湾渡航証）。

### 🔍 深層データマイニング・情報分析
国別特化の「リバースエンジニアリング」ロジックを内蔵し、MRZ オプション領域から隠し情報を抽出：
*   **🇨🇳 中国（CHN）**:
    *   **GBK 隠し氏名解析**: 電子パスポートの暗号化領域から中国語氏名を復元。
    *   **ピンインクロス検証**: MRZ ピンインと中国語氏名を自動比較し、偽造の切断マーカーを検出（改ざん防止ロジック）。
*   **🇩🇪 ドイツ（DEU）**: 身分証の各行政区コード及びシリアル番号ロジックを解析。
*   **🇪🇸 スペイン（ESP）**: DNI 身分証番号を抽出。
*   **🇵🇱 ポーランド（POL）**: PESEL 市民身分番号を抽出。
*   **🇳🇱 オランダ（NLD）**: BSN 市民サービス番号を抽出。
*   **その他**: フランス、ベルギー、スイス、ルーマニア、ブルガリア、ウクライナ、ロシア、北欧四カ国、イスラエルなど20以上の地域の独自ロジック。

### ⚠️ 自動リスク評価
システムは以下のリスクを自動計算・表示：
*   **データ整合性**: 全フィールド Modulus-10 チェックデジット検証。
*   **証明書ステータス**: 有効期限切れアラート、期限切れ間近警告。
*   **論理的脆弱性**: 偽造証明書に共通する論理エラーの検出（例：切断マーカーと残りスペースの不一致、主張する漢字数を構成するのに不十分な母音数など）。

### 💻 ビジュアルインターフェース
*   **ターミナルスタイルビュー**: MRZ 生データをハイライト色分け表示（発行国、氏名、証明書番号、チェックデジット）。
*   **日英二言語対応**: ワンクリック言語切替で国際共同法執行シーンに対応。

## 3. 技術スタック
*   **コアフレームワーク**: React 19
*   **言語**: TypeScript（厳格な型安全性）
*   **スタイリング**: Tailwind CSS（レスポンシブ産業デザイン）
*   **補助ライブラリ**: `pinyin-pro`（中国語氏名ピンインの高精度比較用）

## 4. デプロイと使用方法

### 動作環境
*   モダンブラウザ（Chrome、Edge、Firefox）。
*   インターネット接続不要。

### 操作ガイド
1.  **データ入力**: メインインターフェースの入力ボックスに、スキャナーまたは手動で証明書の MRZ コードを入力（スペースと改行は自動削除）。
2.  **検証実行**: 「検証実行」ボタンをクリックまたは Enter キーを押す。
3.  **レポート表示**:
    *   **ステータスバー**: 検証結果、有効期限ステータス、リスクレベルを表示。
    *   **身元情報**: 解析された氏名、性別、生年月日などを表示。中国パスポートの場合、GBK デコードされた中国語氏名の表示を試行。
    *   **技術パラメータ**: 証明書の詳細タイプを表示（国・形式まで精密）。
    *   **データインサイト**: 基盤となるチェックデジット計算ロジックを表示。

## 5. サポート証明書タイプ一覧（一部）
システムは以下の証明書を正確に識別・分類可能：
*   **大中華圏**: 中華人民共和国普通/電子パスポート、香港特別行政区パスポート、マカオ特別行政区パスポート、香港・マカオ・台湾渡航証。
*   **南北アメリカ**: 米国パスポート/パスポートカード、カナダパスポート、ブラジルパスポート。
*   **ヨーロッパ**: ドイツ、フランス、英国、イタリア、スペイン、オランダ、ポーランド、スイス、ロシア、ウクライナなどのパスポート及び身分証。
*   **アジア太平洋**: 日本、韓国、オーストラリア、ニュージーランド、シンガポール、マレーシア、タイ、インドなど。
